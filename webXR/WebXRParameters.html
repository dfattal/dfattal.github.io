WebXR Projection Control

This example demonstrates how to control SRHydra's projection mode using WebXR's updateRenderState() method.

The projection mode is encoded in the decimal part of the depthFar parameter:

  • Camera Centric (XXX.0): Traditional projection from eye position
  • Display Centric (XXX.1): Projection from display center for better parallax

Note: For reliable operation, this example:

  1. Uses a depth buffer (depthFormat: 'depth-component')
  2. Sets precise far values (10000.0 vs 1200.1)
  3. Calls updateRenderState() at the right time
