Development Setup

  • Clone or download the repository
  • The extension uses Manifest V3
  • Main files: content.js, popup.js, manifest.json
  • Dependencies are included in the libs/ folder

File Structure

Understanding the extension architecture:

  • content.js - Main extension logic and image detection
  • popup.js/html - Extension popup interface
  • libs/ - LIF processing libraries and dependencies
  • shaders/ - WebGL shaders for 3D rendering
  • icons/ - Extension icons in different sizes
